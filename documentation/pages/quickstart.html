10 minutes quickstart

Prerequisite

Angular-Wakanda relies mainly on node.js and gulp, so you have to install it on your computer.

  • node.js download page
  • npm install -g gulp
In this quickstart, we will focus on creating an Angular application with Wakanda Studio, but there are other way to do it, you can refer to Bootstrap an Angular-Wakanda app guide to learn more.

Creating a solution

In this quickstart, we will create a super-simple todo list with Angular-Wakanda.
First of all, create a new solution with Wakanda Studio and leave checked the box Add a blank project to the new solution. Then create an Item dataClass on the data model, located on database/model. When it is done, you can launch the solution by clicking on Run Page.

Our Angular application is located on web/app folder, on Solution Explorer on the left pane. Angular-Wakanda and AngularJS are already installed.

The WakandaManager service

On the scaffolded application when creating a new solution, there is a WakandaManager service avalaible on web/app/scripts/app.js file. This service aimed at giving you a simple way to use your $wakanda instance object everywhere in your application.

It will init $wakanda and exposes it through WakandaManager.$wakanda property. It exposes a promise indicating if $wakanda has finished initializing or not.

You can use this service two different ways :

  • By directly injecting it on a component, putting your code on the callback called when WakandaManager promise is resolved
  • By letting your router resolving the promise and injecting the service when it resolves 

//Injecting WakandaManager directly on a component
app.controller('myController', function ($scope, WakandaManager) {
  WakandaManager.ready().then(function () {
    var $wakanda = WakandaManager.$wakanda;
    var ds = $wakanda.$ds; //ds stands for dataStore, we will see it in the following examples
  });
});

//Injecting with resolve property of a route. Here with ngRoute
app.config(function ($routeProvider) {
  $routeProvider
    .when('/', {
      templateUrl: 'home.html',
      controller: 'homeController',
      resolve: {
        manager: function (WakandaManager) {
          return WakandaManager.ready();
        }
      }
    });
})
.controller('homeController', function($scope, manager) {
  var $wakanda = manager.$wakanda;
});

In this quickstart, we will directly inject WakandaManager service on controller. But for a real application, it is strongly advised to resolve it on router.
