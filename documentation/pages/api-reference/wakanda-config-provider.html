$wakandaConfig provider

$wakandaConfig is a useful provider to access the hostname of the Wakanda Server the application is working with.

On application configuration

setHostname(hostname)

Set the hostname to the provided value. All REST calls from Angular-Wakanda will now be sent to this server. Be careful about CORS (Cross-Origin Resource Sharing) issues. 

angular.module('application', ['wakanda'])
  .config(function ($wakandaConfigProvider) {
    $wakandaConfigProvider.setHostname('http://webserver.com:1136');
  });

Everywhere on the app

getHostname()

Return the hostname given on application config, or undefined if not set. 

app.controller('FooController', function ($scope, $wakandaConfig) {
  $wakandaConfig.getHostname(); //http://webserver.com:1136
});
